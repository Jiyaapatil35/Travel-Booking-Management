# Travel Booking Management System

**Semester:** 3rd | **Duration:** Sept’24 – Jan’25  
**Team Size:** 4 Members | **Language:** C++

## Project Overview
The **Travel Booking Management System** is a comprehensive console-based application that allows users to book and manage travel-related services.  
The system is implemented in **C++** and includes three main modules:

1. **Bus, Train, and Flight Booking**  
2. **Hotel Booking** *(My Contribution)*  
3. **Vehicle Renting**

The system supports searching, booking, cancellation, and viewing booking history for tickets, hotels, and vehicles.

## Modules & Features

### **1. Bus, Train, and Flight Booking**
- Search for available tickets  
- Book tickets  
- Cancel bookings  

### **2. Hotel Booking** *(My Contribution)*
- User and admin dashboards  
- Add or delete hotels *(admin only)*  
- Search for available hotels  
- Book hotel rooms  
- Cancel hotel bookings  
- View booking history  

### **3. Vehicle Renting**
- Search for available vehicles  
- Book vehicle rentals  
- Cancel vehicle rentals  
- View rental history
